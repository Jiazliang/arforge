# Client-Server Example

This project focuses on client-server communication.

It highlights:

- a client-server interface with arguments, return types, and possible errors
- a client SWC that performs synchronous and asynchronous calls
- a server SWC with `operationInvokedEvents` bound to the provided client-server port
- a simple supporting sender-receiver signal so the client example also shows mixed port usage

Entry point: `examples/features/client_server/autosar.project.yaml`
